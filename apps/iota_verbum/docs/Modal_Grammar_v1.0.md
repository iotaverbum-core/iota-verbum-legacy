# Modal Grammar v1.0 (□ necessity, ◇ enactment, Δ effect)

## 1. Operators and Engine Fields
- □ (Necessity / Identity): engine **Subject** (agent identity; covenantal being)
- ◇ (Enactment / Act): engine **Act** (what is done)
- Δ (Effect / Result): engine **Result** (what comes to be)

Frame shape: **Subject → Act → Objects → Scope/Time → Result**  
Operator mapping:  □Subject   ◇Act   ΔResult
