## Description
EDEN helps Christians bring ordinary moments into honest prayer before God. It uses deterministic posture analysis and deterministic validation/repair around responses.

AI sharing is explicit opt-in. If the user selects `Not now`, EDEN remains fully usable in Local mode using deterministic templates only.

EDEN does not claim divine authority. EDEN is not therapy, not medical care, not crisis care, and not prophecy.
